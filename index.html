quickScript

QuickScript is a simple and easy to learn special-purpose language which can be used to design artificial conversational agents and other programs that involve textual dialogue between humans and computers.
Up for making your own chatbot?

Get QuickScript

Features

{easy to learn, 

easier to write}

QuickScript is special because it is simple, coherent and open source. Hence, it fades the barrier between a normal computer user and a creative botmaster (a person who creates chatbots). It is intended to generate wider interest in the field of AI and virtual conversational agents. With QuickScript, you can create interactive textual systems ranging from simple dictionaries to specialized chatbots like child's play. The features of QuickScript are inspired by AIML, but the way of writing code has been even more simplified! Apart from providing all the basic gear that one would appreciate when coding his/her own chatbot, it also eases the process to such an extent that anyone can playfully learn this art in minutes.

The QuickScript Engine, some sample databases (QS files), the source code and complete documentation are bundled in the following ZIP file. You can code and run your files in the QS Engine, which is written in C and compiled successfully on Code::Blocks 13.12 (MinGW GCC Compiler 4.8). For complete info on how to write and run QuickScript code and make your own chatbots, read the documentation file "Coding With QuickScript".
